# is-simulation
> Check whether the code is running in simulation

## Install
```sh
$ npm install @open-screeps/is-simulation
```

## Usage
```typescript
import { isSimulation } from '@open-screeps/is-simulation';

if (isSimulation()) {
  console.log('Code loaded in simulation.');
}
```
